Privacy Policy

GhostReel  ·  Last updated: March 2026

GhostReel is a personal video automation tool used solely by its owner to generate and schedule short-form video content for posting to their own social media accounts. This Privacy Policy describes how data is handled in connection with our use of the TikTok Content Posting API.

1. Data We Access

When connecting to TikTok, GhostReel requests the video.publish scope in order to upload videos to the authorised TikTok account. This grants access to:

  • The TikTok user's access token and Open ID (used solely to authenticate API requests)
  • Basic creator information required by the TikTok API (e.g. maximum video duration)

2. How We Use Your Data

Data accessed through the TikTok API is used exclusively to post video content to the authorised account on behalf of the account owner. Specifically:

  • Access tokens are stored locally on the user's own machine, encrypted using industry-standard symmetric encryption (Fernet/AES-128)
  • Tokens are used only to authenticate video upload requests to TikTok's API
  • No data is transmitted to any third-party server
  • No data is sold, rented, or shared with any third party

3. Data Storage

All data — including OAuth tokens, generated video files, and application settings — is stored locally on the user's own machine. GhostReel does not operate any cloud servers or databases. There is no central storage of any user data.

4. Data Retention

OAuth tokens are retained locally until the user disconnects their TikTok account via the application settings, at which point all stored tokens are deleted. Generated video files are stored locally and may be deleted by the user at any time.

5. Third-Party Services

GhostReel integrates with the following third-party APIs to provide its core functionality:

  • TikTok Content Posting API — for publishing video content
  • Anthropic Claude API — for AI script generation (no personal data is sent)
  • Fish Audio API — for text-to-speech voice generation (no personal data is sent)
  • Pexels API — for stock footage (no personal data is sent)

Each service is subject to its own privacy policy and terms of service.

6. Children's Privacy

GhostReel is a personal tool intended for adult use only. It does not knowingly collect data from or target children under the age of 13.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. The date at the top of this page reflects the most recent revision. Continued use of the application constitutes acceptance of the updated policy.
